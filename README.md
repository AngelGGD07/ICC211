CSTI-1910-6337: Algoritmo Clásico y Estructura de Datos

Repositorio académico que contiene el material práctico, las asignaciones y los proyectos desarrollados para la asignatura Algoritmo Clásico y Estructura de Datos. Este espacio sirve como colección centralizada de implementaciones de estructuras de datos, análisis de algoritmos y resolución de problemas lógicos.

Herramientas y Entorno
El código y los proyectos contenidos en este repositorio fueron desarrollados bajo el siguiente entorno:
-IDE Principal: IntelliJ IDEA
-Lenguaje: Java
-Control de Versiones: Git & GitHub

Estructura del Repositorio
El contenido se encuentra organizado categóricamente para facilitar la navegación y el estudio de los distintos temas abordados:

Contenido de los Módulos

Tareas
Colección de asignaciones semanales enfocadas en la lógica de programación, análisis de complejidad algorítmica (Big O) y fundamentos teóricos de las ciencias de la computación.

Prácticas
Implementaciones funcionales de las estructuras de datos y algoritmos estudiados durante el curso. Incluye:
Estructuras Lineales: Listas Enlazadas (Simples, Dobles, Circulares), Pilas (Stacks) y Colas (Queues).
Estructuras No Lineales: Árboles (Binarios, AVL, B-Trees) y Grafos.
Algoritmia: Métodos de ordenamiento (Sorts) y algoritmos de búsqueda eficiente.

Exámenes
Archivo de las evaluaciones prácticas y teóricas realizadas, incluyendo sus respectivas correcciones y soluciones optimizadas.

Proyecto Final
Desarrollo de una aplicación integradora que aplica los conocimientos adquiridos en la materia para resolver una problemática específica, demostrando el uso eficiente de estructuras de datos y buenas prácticas de programación.
